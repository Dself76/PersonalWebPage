PROJECTS I HAVE COMPLETED AND ARE WORKING ON

Project Name: Simple Password Manager

Language: C++

Project Overview:
The Simple Password Manager is a C++ application designed to securely store and manage user passwords. This project aims to provide a basic yet effective solution for handling password storage with a focus on simplicity and security. It demonstrates the fundamental concepts of file handling, encryption, and user authentication in C++.

Features:
1. Password Storage: Securely store passwords with associated usernames and services.
2. Encryption: Implement basic encryption techniques to protect stored passwords.
3. User Authentication: Ensure that only authorized users can access the stored passwords.
4. Command-Line Interface: Simple and intuitive command-line interface for user interactions.

Project Structure:
- main.cpp: The main entry point of the application.
- PasswordManager.cpp: Contains the core functionality of the password manager, including methods for adding, retrieving, and encrypting passwords.
- PasswordManager.h: Header file defining the PasswordManager class and its methods.
- PasswordManager.sln: Visual Studio solution file for the project.
- PasswordManager.vcxproj: Visual Studio project file.
- PasswordManager.vcxproj.filters: Filters file for organizing the project files in Visual Studio.
- PasswordManager.vcxproj.user: User-specific settings for the Visual Studio project.

Development Environment:
- Developed using Visual Studio, leveraging its robust development tools and debugging features.
- The project is configured to build and run in a Debug configuration on a 64-bit Windows environment.

How to Run:
1. Open the PasswordManager.sln solution file in Visual Studio.
2. Build the project using the provided solution configuration.
3. Run the compiled executable to start the password manager.

Usage:
- Add Password: Store a new password with the associated username and service name.
- Retrieve Password: Retrieve stored passwords by providing the service name.
- Encrypt/Decrypt: Basic encryption and decryption functionalities to secure stored passwords.

This project helped me with the following Learning Outcomes:
- Understanding of file handling in C++.
- Implementation of basic encryption techniques.
- Experience with user authentication mechanisms.
- Familiarity with developing and debugging C++ applications in Visual Studio.
